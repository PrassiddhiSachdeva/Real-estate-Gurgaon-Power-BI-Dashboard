# Real Estate Dashboard - Power BI Project

## Overview
This project analyzes real estate trends in Gurgaon using Power BI. It provides insights into pricing, supply hotspots, and geographic clustering.

## Problem Statement
- Identify high-activity real estate zones
- Analyze price variations across sectors
- Help investors make data-driven decisions

## Project Contents
- Dataset used for analysis
- Power BI dashboard (.pbix file)
- Project report
- Presentation (PPT)
- Dashboard screenshots
- Video walkthrough

## Key Insights
- High-density property clusters indicate active markets
- Price per sq. ft varies significantly across sectors
- Certain zones show strong investment potential

## Tools Used
- Power BI
- Excel / CSV datasets
